- Установите необходимые библиотеки из файла requirements.txt. Команда консоли: pip install -r requirements.txt
- Установите ChromeDriver и добавьте в переменную PATH. Скачать chromedriver: https://chromedriver.chromium.org. Скачанная версия хромдрайвера должна соответствовать установленному на вашем ПК браузеру Google Chrome.
- Запустите //pyfiles//first_start.py. Его нужно запустить один раз для создания необходимых файлов.

Он создаст:
- 'inf.yml': должен содержать ваш токен sms-activate, название страны для аренды номера и пол будущих аккаунтов
Пример наполнения файла:

token: 545783907509287hsbdf89
country_name: Россия
sex: MALE

- 'proxies.yml': будет содержать одну вашу проксю
Пример наполнения файла:

proxy: Login:pass:4.303.23.298:8000

- 'login, pass.txt': сюда будет сыпаться информация об авторегах
- 'sorted_prices.yml': при использовании фичи автоподбора самого дешевого номера, каждый раз будет переписывать файл актуальным сортированным списком цен 

Обращайте внимание на консоль, она попросит прокси в формате login:pass:ip:port, если прокси открытый: ip:port. Если не хотите использовать прокси, просто нажмите Enter (при отсутствии прокси файл //your_files//proxies.yml должен быть пустым!).

Также от вас потребуется выбрать страну, номер которой вы будете арендовать. Советую выбирать Россию. Также существует фича автоподбора страны с самым дешевым номером аренды, но она пока бета. Если хотите попробовать, в файлике 'inf.txt' напишите country_name:False вместо country_name:*название страны*
